Negative Binomial

Information

The negative binomial distribution is used to model the number of trials in a sequence before a specified number of successes occur.

Probability mass function

\(P_x = {x-1 \choose k-1}(1-p)^{x-k}(p)^k\)

\(x\) : the number of trials
\(k\) : the number of successes
\(p\) : probability of success on a single trial
