Hypergeometric

Information

The hypergeometric distribution is used to model proportions of a given item in samples.

Probability mass function

\(P_x = \frac{{K \choose x}{N-K\choose n-x}}{N \choose n}\)

\(x\) : the number of observed successes
\(N\) : the population size
\(K\) : the number of success states in the population
\(n\) : the number of draws
